# PRD Context: 揪Gym (JoyGym)

## Executive Summary

揪Gym (JoyGym) 是一個專注於社交與重量訓練記錄的 app，核心價值是：

1. 針對不同健身房與器材差異，提供可比較紀錄（gym-aware）
2. 用 template 加速訓練，同時保留當日調整彈性
3. 透過打卡、好友動態、愛心互動形成訓練回饋循環
4. 透過儀表板分析重訓狀態、進步幅度，藉此找出適合自己的練習方法。
5. 透過圖表與成就系統回饋長期進步

## Problem Statement

1. 同動作在不同 gym 的器材差異下，重量不可直接比較
2. 現場器材占用常需臨時換動作
3. 缺乏圖表分析
4. 長期缺乏可追蹤進步與社交激勵
5. 沒有持續打卡與社交回饋時，訓練習慣難以維持

## Target Users

1. 社交訓練者
2. 多場館訓練者
3. 課表化訓練者
