# G-10 Share Card

## Goal Statement

使用者可將打卡或訓練成果產生分享卡片。

## Requirement

系統必須提供分享卡模板選擇與渲染輸出。

## In Scope

1. 打卡/訓練摘要分享卡
2. 模板清單
3. 圖片輸出

## Out Scope

1. 社群平台一鍵發布 API 整合
2. 自訂模板編輯器

## Acceptance Criteria

- [ ] Given 使用者選擇模板，When 產生分享卡，Then 回傳可用圖片結果。
- [ ] Given 免費方案使用者，When 選擇 Pro 模板，Then 顯示升級限制。
