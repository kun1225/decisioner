# G-11 Progress Charts

> **Phase:** 1.0 | **Status:** Not Started
> **Specs:** [API &sect;5.8](../specs/5-api-design.md) &middot; [Data](../specs/6-data-model.md)

## Goal Statement

使用者可查看免費版核心進步圖表（max weight / volume）。

## Requirement

系統必須提供單動作 Max Weight 與 Volume 趨勢圖。

## In Scope

1. Max Weight chart
2. Volume chart
3. 基本時間區間查詢

## Out Scope

1. 肌群維度聚合圖
2. e1RM 與 adherence 圖

## Frontend Elements

1. Progress 頁動作選擇器
2. 圖表切換控制（Max Weight/Volume）
3. 時間區間切換器
4. 無資料狀態元件

## Acceptance Criteria

- [ ] Given 使用者有多次訓練，When 開啟 progress，Then 可看到 max weight 趨勢。
- [ ] Given 使用者有多次訓練，When 切換 volume，Then 可看到 volume 趨勢。
- [ ] Given 使用者切換不同動作，When 請求圖表資料，Then 圖表應更新為該動作的趨勢。

## Tasks

- [ ] Max weight / volume chart API（time range query）
- [ ] Progress 前端頁面（動作選擇 + 圖表切換 + 時間區間）
