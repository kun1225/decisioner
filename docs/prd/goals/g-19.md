# G-19 Global Site Header

## Goal Statement

使用者在主要頁面都可透過一致的 header 快速導航並辨識目前登入狀態。

## Requirement

系統必須提供全站共用 header，包含品牌、導覽與登入狀態相關入口。

## In Scope

1. 全站 header 版面結構
2. 桌面版主導覽連結
3. 手機版導覽入口（menu trigger）
4. 已登入/未登入狀態下的 header 動作區

## Out Scope

1. 多層 mega menu
2. 個人化導覽推薦

## Frontend Elements

1. Header 容器
2. Brand/Logo 區塊
3. 主導覽連結群組
4. Auth action 區塊（登入、註冊、頭像）
5. Mobile menu trigger

## Acceptance Criteria

- [ ] Given 使用者進入任何主要頁面，When 畫面載入完成，Then 可看到一致的 header 結構。
- [ ] Given 使用者未登入，When 查看 header，Then 顯示登入/註冊入口。
- [ ] Given 使用者已登入，When 查看 header，Then 顯示使用者狀態入口（如頭像或帳號選單入口）。
- [ ] Given 使用者在手機寬度瀏覽，When 查看 header，Then 顯示可操作的 mobile menu trigger。
