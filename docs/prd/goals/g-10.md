# G-10 Share Card

> **Phase:** 1.0 | **Status:** Not Started
> **Specs:** [API &sect;5.9](../specs/5-api-design.md) &middot; [Data](../specs/6-data-model.md)

## Goal Statement

使用者可將打卡或訓練成果產生分享卡片。

## Requirement

系統必須提供分享卡模板選擇與渲染輸出。

## In Scope

1. 打卡/訓練摘要分享卡
2. 模板清單
3. 圖片輸出

## Out Scope

1. 社群平台一鍵發布 API 整合
2. 自訂模板編輯器

## Frontend Elements

1. 打卡完成頁/訓練完成頁的分享入口
2. 分享模板清單（含免費/Pro 標記）
3. 分享卡預覽區
4. 匯出按鈕與升級提示

## Acceptance Criteria

- [ ] Given 使用者選擇模板，When 產生分享卡，Then 回傳可用圖片結果。
- [ ] Given 免費方案使用者，When 選擇 Pro 模板，Then 顯示升級限制。
- [ ] Given 使用者在打卡完成頁或訓練完成頁操作分享，When 進入分享流程，Then 可直接選擇模板並輸出卡片。

## Tasks

- [ ] Share card templates API + DB
- [ ] Share card render API（圖片產生）
- [ ] 分享卡前端流程（模板選擇 + 預覽 + 匯出）
