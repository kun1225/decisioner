# G-17 Profile Customization

> **Phase:** 2.1 | **Status:** Not Started
> **Specs:** [API &sect;5.9](../specs/5-api-design.md) (future)

## Goal Statement

Pro 使用者可客製個人頁主題與展示方式。

## Requirement

系統必須提供個人頁客製化設定與儲存。

## In Scope

1. 主題/封面設定
2. 展示區塊排序
3. 客製化儲存

## Out Scope

1. 版型市場
2. 客製化 CSS 編輯器

## Frontend Elements

1. 個人頁客製化設定面板
2. 主題/封面選擇器
3. 展示區塊排序元件
4. Pro 限制提示

## Acceptance Criteria

- [ ] Given Pro 使用者調整個人頁設定，When 儲存成功，Then 公開頁顯示新設定。
- [ ] Given 非 Pro 使用者，When 嘗試儲存客製化，Then 顯示方案限制。

## Tasks

- [ ] Profile customization settings API + DB
- [ ] 個人頁客製化前端面板
