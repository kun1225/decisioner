# G-14 Weekly Challenge

## Goal Statement

使用者可加入每週挑戰並追蹤目標進度。

## Requirement

系統必須支援 weekly challenge 的加入與進度更新。

## In Scope

1. 週挑戰清單
2. 加入挑戰
3. 進度累積

## Out Scope

1. 挑戰商城
2. 跨月挑戰賽季系統

## Frontend Elements

1. 週挑戰清單頁
2. 挑戰加入按鈕
3. 挑戰進度條元件
4. 挑戰詳情面板

## Acceptance Criteria

- [ ] Given 有 ACTIVE 挑戰，When 使用者加入，Then 建立參與紀錄。
- [ ] Given 使用者完成對應行為，When 系統更新進度，Then 挑戰進度正確累加。
