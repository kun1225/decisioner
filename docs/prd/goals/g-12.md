# G-12 Pro Analytics

> **Phase:** 1.3 | **Status:** Not Started
> **Specs:** [API &sect;5.8](../specs/5-api-design.md) &middot; [Data](../specs/6-data-model.md)

## Goal Statement

Pro 使用者可使用進階分析（e1RM、肌群週訓練量、adherence、RPE/RIR）。

## Requirement

系統必須提供 Pro 分析資料與記錄欄位。

## In Scope

1. e1RM 趨勢
2. Weekly muscle volume
3. Weekly adherence
4. Set-level RPE/RIR 記錄

## Out Scope

1. AI 分析建議
2. 教練端報告匯出

## Frontend Elements

1. Pro Analytics 頁分區（e1RM/肌群/adherence）
2. Set 輸入中的 RPE/RIR 欄位
3. Pro 限制提示區塊
4. 每週目標設定表單

## Acceptance Criteria

- [ ] Given Pro 使用者，When 開啟進階分析頁，Then 可讀取 e1RM/肌群/adherence 資料。
- [ ] Given 非 Pro 使用者，When 請求進階分析，Then 回傳方案限制。
- [ ] Given Pro 使用者記錄 set，When 儲存資料，Then 可同時保存 `rpe` 與 `rir`。
- [ ] Given Pro 使用者設定每週訓練目標，When 查詢 adherence，Then 分母應使用該週目標值。

## Tasks

- [ ] e1RM 計算邏輯 + chart API
- [ ] Weekly muscle volume API
- [ ] Weekly adherence API + training goal CRUD
- [ ] Set RPE/RIR 欄位（API + DB migration）
- [ ] Pro analytics 前端頁面
