# 1. Tech Stack

## 1.1 Runtime

1. Frontend: React 19 + TanStack Start + Vite
2. Backend: Node.js + Express
3. Database: PostgreSQL + Drizzle ORM

## 1.2 UI and Styling

1. shadcn (Base UI)
2. Tailwind CSS v4
3. CVA + clsx + tailwind-merge

## 1.3 Data and Routing

1. TanStack Router
2. TanStack Query

## 1.4 Auth and Security

1. JWT access token
2. Refresh token rotation (HttpOnly cookie)
3. bcrypt password hashing
4. Google idToken login

## 1.5 Storage and Media

1. S3 pre-signed upload
2. CDN-friendly asset URL model

## 1.6 Testing and Quality

1. Vitest + Testing Library
2. ESLint + TypeScript typecheck
3. Playwright (E2E)
